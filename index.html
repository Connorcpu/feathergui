What is FeatherGUI?

Download

FeatherGUI SDK v0.1.0

FeatherGUI is a GUI abstraction layer that seperates logic from implementation. Instead of abstracting over a hardcoded set of existing GUI APIs, FeatherGUI implements an abstraction layer you then implement in your own engine by linking a few functions. All you need is a way to draw images, text, and lines, and featherGUI will do the rest!

While this makes FeatherGUI great for games, it also supports using native GUI APIs, allowing you to build just one layout and use that same layout across all platforms. Because you control the implementation of FeatherGUI, if an existing implementation doesn't exist for your target platform, it's trivial to make one yourself. Your GUI toolkit should work for you, not the other way around. Take back control of how your GUI behaves.
